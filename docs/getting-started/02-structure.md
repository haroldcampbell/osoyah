# 02 Structure

High-level folders
- `client/`: Angular UI.
- `server/`: FastAPI backend (future).
- `specs/`: Requirements and milestones.
- `docs/`: Principles, process, decisions, architecture.
- `scripts/`: Helper scripts.
- `designs/`: Mockups and visual references.
- `session-hand-offs/`: Session summaries.
- `out/`: Generated artifacts.

Key files
- `AGENTS.md`: Guardrails and safety rules.
- `docs/process.md`: Workflow entry point.
- `docs/principles.md`: Engineering/product principles.
- `docs/decisions.md`: Current decisions.
- `specs/milestone.md`: Milestone checklist.
- `session-hand-off.md`: Hand-off index and rules.
