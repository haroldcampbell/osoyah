# 06 Standards + Glossary

Naming + commits
- Specs: `S00X-...` in `specs/M00X-*/`.
- Milestones: `specs/M00X-*/milestone.md`.
- Hand-offs: `yyyy-mm-dd-nn-title.md` (no `S00X`/`M00X`).
- Commits: `feat (M00X-S00X): short description`.

Glossary
- Board: collection of lists.
- List: ordered column of card IDs.
- Card: global entity shared across boards.
- Spec: scoped requirement doc.
- DoD: Definition of Done.
- Hand-off: session summary.
