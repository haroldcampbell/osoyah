# M-future-000-Responsive: Adaptive Layouts

Conform to `docs/principles.md`.

## Summary
Plan a future milestone focused on making the application fully responsive across mobile, tablet, and desktop.

## Goal
Ensure the board, lists, cards, and side panel adapt gracefully to smaller screens without losing usability.

## Scope
In-scope:
- Responsive layouts and spacing across the board UI.
- Touch-friendly interactions and hit targets.
- Mobile-first adjustments for the card detail panel.
Out-of-scope:
- New features beyond layout/responsiveness.
- Visual redesign beyond responsive adjustments.
