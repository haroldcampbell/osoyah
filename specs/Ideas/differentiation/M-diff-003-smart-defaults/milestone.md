# M-diff-003-Smart-Defaults: Templates + Guided Onboarding

Conform to `docs/principles.md`.

## Summary
Provide opinionated templates and a guided onboarding flow so new teams can reach first value quickly without configuration overhead.

## Goal
A new user can create a usable workspace/board in minutes with minimal decisions.

## Scope
In-scope:
- Template catalog for common agency workflows.
- Guided onboarding that uses templates and teaches key actions.

Out-of-scope:
- Custom workflow builders.
- Long, modal-heavy tutorials.
