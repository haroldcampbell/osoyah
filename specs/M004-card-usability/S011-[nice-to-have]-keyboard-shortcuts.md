# S011-[Nice-to-Have] Keyboard Shortcuts: Fast Navigation

Conform to `principles.md`.

## Summary
Add keyboard shortcuts for common actions (open card, edit title, add comment).

## Goal
Enable faster workflows for power users without sacrificing discoverability.

## Non-goals
- Full keyboard-only navigation of the entire board.
- Custom shortcut configuration.

## Definition of Done
- [ ] Keyboard shortcuts exist for open card, edit title, and add comment.
- [ ] Shortcuts are discoverable via visible hints or tooltips.
- [ ] Shortcuts avoid conflicts with browser defaults.
- [ ] Acceptance tests pass.

## Acceptance tests (exact commands + expected artifacts/output)
- `npm run lint` passes.
- `npm run test` passes.
